# Reports - Rapports d'analyse

Ce répertoire contient les rapports d'analyse et résultats intermédiaires.

## Organisation

- Rapports générés automatiquement depuis notebooks ou scripts
- Formats : PDF, HTML, Markdown
- Nommer avec date : `YYYY-MM-DD_titre_rapport.pdf`

## Contenu typique

- Analyses exploratoires
- Statistiques descriptives
- Visualisations
- Résultats intermédiaires

## Notes

Les fichiers PDF et HTML ne sont pas versionnés (voir `.gitignore`).
Seuls les scripts/notebooks qui génèrent ces rapports sont versionnés.
